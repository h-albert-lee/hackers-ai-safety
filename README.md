# AI Agent 보안 (해커스 HRD)

## 강의 개요
이 프로젝트는 해커스 HRD에서 진행한 "AI Agent 보안" 강의의 실습 및 학습 자료입니다. AI 에이전트의 보안 위협, 안전 설계, 레드팀 실습, 안전 자동화 등 최신 AI 보안 트렌드와 실전 대응 방법을 다룹니다.

## 목적
- AI 에이전트의 보안 위협과 대응 방안 이해
- 안전한 AI 시스템 설계 및 구현 실습
- 레드팀/블루팀 관점에서 AI 시스템 공격·방어 경험
- 최신 AI 보안 도구 및 자동화 활용법 습득

## 강의 목차 및 실습 자료
- 02-safety-classifier.ipynb: 안전 분류기 설계 및 실습
- 03-preference-reward.ipynb: 선호도 기반 보상 설계
- 04-constitutional-ai.ipynb: 헌법적 AI 설계
- 05-red-team.ipynb: 레드팀 공격 실습
- 06-jailbreak.ipynb: AI 시스템 탈옥 공격 실습
- 07-advanced-red-team.ipynb: 고급 레드팀 기법
- 08-guardrails-moderation-api.ipynb: Guardrails 및 Moderation API 실습
- 09-safety-designing.ipynb: 안전 설계 원칙
- 10-agent-structure-smolagent.ipynb: 에이전트 구조 및 구현
- 11-agent-threats.ipynb: 에이전트 위협 분석
- 12-safety-automation.ipynb: 안전 자동화 실습
- 13-secure-agent-design.ipynb: 안전한 에이전트 설계
- 14-automated-red-teaming.ipynb: 자동화된 레드팀 실습
- 15-secure-code-agent.ipynb: 안전한 코드 에이전트
- 16-safety-report.ipynb: 안전성 보고서 작성

## 활용 방법
1. 각 ipynb 파일을 Jupyter Notebook에서 실행하며 실습 진행
2. 실습 전 환경 변수(OPENAI_API_KEY, OPENROUTER_API_KEY 등) 설정 및 크레딧 충전 필요
3. 각 노트북 내 안내에 따라 실습 및 코드 수정
